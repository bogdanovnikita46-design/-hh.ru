# Высшее образование больше не обязательно: как меняются требования к креативным специалистам

## Синопсис

Проект посвящён анализу трансформации требований работодателей к уровню образования в креативных профессиях в России. На основе данных вакансий исследуется, насколько важно наличие высшего образования по сравнению с практическими навыками, опытом и портфолио. Также рассматривается распределение профессий креативных индустрий по России, уровень зарплат и важность опыта работы.

Анализ строится на данных с hh.ru, собранных через API и парсинг. В центре внимания — изменение структуры требований работодателей и выявление тренда на снижение значимости формального образования в креативных индустриях. Вакансии былы собраны на за февраль-март 2026года. Было проанализировано 5647 вакансий креативных индустрий со всей России

---

## Актуальность

Изменения на рынке труда напрямую влияют на образовательные траектории и карьерные стратегии. В креативных профессиях (дизайн, маркетинг, медиа, IT-креатив) всё чаще наблюдается смещение от формальных требований (диплом) к прикладным навыкам, опыту и портфолио

Это важно, потому что:
- влияет на выбор образования и инвестиций в него;
- меняет барьеры входа в профессию;
- отражает структурные изменения рынка труда;
- актуально для студентов, абитуриентов, молодых специалистов и работодателей.

## Исследовательские вопросы и ответы:

- Какие альтернативные требования (опыт, портфолио, навыки) чаще всего указываются вместо образования?
Анализ показал, что уровень образования вовсе ну указывался ни в одной из обработанных вакансий. Главным требованием был опыт работы, это значение было указано в каждой вакансии, упоминание слова "портфолио" в тексте вакансии незначительное — 0,3%. Однако, сворее всего, у портфолио такой малый показатель из-за специфики платформы hh.ru, работадатели знают, что чаще всего у каждого, человека, который ищет работу на жтом портале в профиле есть ссылки на портфолио и резюме, кроме того предполагается, что человек, претендующий на вакансию, в которой указан определённый опыт, сможет доказать этот самый рабочий опыт с помощью портфолио 

- Каково распределение рынка креативных индустрий по России?
Москва концентрирует 50.8% всех вакансий. Санкт-Петербург — 11.8%. Остальные города суммарно занимают 37.4%. Это говорит о высокой централизации рынка креативных профессий.

## Данные



### Источник

Основной источник данных — hh.ru (HeadHunter), крупнейший портал вакансий в России.

Используются:
- API hh.ru
- результаты поисковых запросов по ключевым словам (креативные профессии)

### Сбор данных

Данные собираются с помощью Python-скрипта:
- обращение к API `/vacancies`
- получение детальной информации по каждой вакансии
- фильтрация по ключевым словам (дизайн, маркетинг, медиа и др.)

### Ключевые поля

- `id` — идентификатор вакансии  
- `name` — название вакансии  
- `area` — регион  
- `employer` — работодатель  
- `description` — текст вакансии  
- `key_skills` — навыки  
- `experience` — уровень опыта  
- `employment`, `schedule` — формат работы  
- `salary` — зарплата  

Дополнительно извлекаются признаки из текста:
- упоминание высшего образования  
- отсутствие требования образования  
- наличие требования портфолио  
- упоминание опыта  

### Очистка данных

- удаление дубликатов по `id`
- нормализация вложенных структур (JSON → таблица)
- извлечение признаков из текста (keyword-based анализ)
- объединение данных в единый датафрейм
